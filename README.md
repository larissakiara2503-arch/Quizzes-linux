# 🐧 Linux Quiz

Quiz interativo de terminal com 50 questões sobre comandos Linux essenciais.

## Como usar

Baixe o arquivo `linux-quiz.html` e abra no navegador. Sem instalação necessária.

## Funcionalidades

- Terminal simulado com output fictício realista
- 50 questões em 13 categorias (arquivos, rede, processos, permissões, git e mais)
- Feedback imediato com acertos e erros
- Dark mode com visual estilo terminal retrô
- Responsivo para mobile (iOS e Android)

## Tecnologias

HTML · CSS · JavaScript Vanilla
